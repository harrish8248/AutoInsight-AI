# 🚀 AutoInsight AI – Automated Data Analysis Platform

## 📌 Overview

AutoInsight AI is an end-to-end data analysis platform that automates the process of data cleaning, exploration, and insight generation. It is designed to reduce the manual effort required in data preprocessing and help users quickly extract meaningful insights from raw datasets.

---

## 🎯 Problem Statement

Data analysis is often time-consuming and requires strong technical expertise. Many users struggle with:

* Cleaning messy datasets
* Performing exploratory data analysis
* Extracting meaningful insights

AutoInsight AI solves this by automating the entire pipeline, making data analysis faster, easier, and more accessible.

---

## ✨ Key Features

* 📂 Upload CSV datasets
* 🧹 Automatic data cleaning (handling missing values, duplicates)
* 📊 Exploratory Data Analysis (EDA)
* 📈 Data visualization for better understanding
* 🤖 AI-powered insight generation
* 🔗 Backend API for scalable processing

---

## 🏗️ System Architecture

```
Frontend (React)

```

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/harrish8248/AutoInsight-AI.git
cd AutoInsight-AI
```

---

### 2. Backend Setup

```bash
cd backend
pip install -r requirements.txt
python main.py
```

---

### 3. Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

---

## ▶️ Usage

1. Start backend server
2. Start frontend application
3. Upload a dataset (CSV format)
